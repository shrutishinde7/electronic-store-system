## Electronic Store System
A web-based system for electronic store operations.

**Tech Stack:** Core Java, JSP, Servlets, JDBC, Oracle SQL

**Features:**
- Product management with role-based access control (RBAC)
- User authentication and transaction tracking
- 99.9% data accuracy with Oracle SQL backend

**How to run:**
1. Clone the repo
2. Import into Eclipse IDE
3. Configure Oracle DB connection in JDBC config
4. Deploy on Apache Tomcat
